基础流程

核心流程​

问卷的基础流程如下：

登录注册​

官方不提供默认账号，可以自行注册账号后进行登录使用：

创建问卷​

创建​

在问卷列表页选择创建问卷。

不同类型的问卷有不同的默认内容。

协作​

给问卷添加协作者，基于不同权限共同管理问卷：

空间​

通过增加空间成员进行问卷共享，便于团队协作。

问卷设计​

内容编辑​

提供问卷搭建的可视化页面，允许用户通过拖拽、点击等简单操作来设计问卷：

1、题型选择区：支持输入类、选择类、矩阵题、高级题等多种选择，点击即可添加题目。各题型功能请参考题型介绍。

2、题目大纲：已添加题目的大纲展示。

3、题型编辑区：问卷可视化操作，可以对题目和选项的内容、顺序进行操作。

4、题型配置区：对题目进行配置操作。

5、公共操作区：包含协作、历史、保存、预览等问卷基础内容操作。

问卷设置​

问卷设置页，用户可以配置问卷的规则。

皮肤设置​

皮肤设置页主要针对 C 端的展示进行管理。

结果页设置​

设置 C 端提交问卷后的展示内容。

逻辑设置​

显示逻辑​

支持条件逻辑，如某些题目只有在满足特定条件时才会显示，根据受访者的回答引导他们回答不同的题目。

问卷投放​

发布​

问卷设计完成后进行发布，生成 C 端问卷。

投放​

生成问卷链接或二维码，方便在网站、社交媒体或通过电子邮件等方式分发。

数据分析​

在线分析​

提供各种统计图表，如饼图、柱状图、折线图等，帮助用户直观地理解数据。

导出​

支持数据导出功能，以便于在专业的统计分析软件中进一步分析。
